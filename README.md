# Vietnamese Coffee Recommender App

A starter React/Vite prototype for an AI-personalized Vietnamese coffee subscription business.

## Features

- Customer taste quiz
- Coffee product database
- Rule-based recommendation engine
- Ranked product recommendations
- Subscription-style best match
- Customer feedback/rating capture
- Clean responsive interface

## How to run

1. Install Node.js
2. Open this folder in Terminal
3. Run:

```bash
npm install
npm run dev
```

Then open the local URL shown in the terminal.

## How to expand this into a real product

Recommended next steps:

1. Replace the hard-coded coffee list with a database such as Supabase, PostgreSQL, or Airtable.
2. Store customer profiles and feedback ratings.
3. Add authentication.
4. Add Shopify, Stripe, or Recharge for checkout/subscriptions.
5. Replace the rule-based score with a machine learning model after you collect enough user ratings.
6. Add admin tools for managing coffee inventory and roast profiles.
